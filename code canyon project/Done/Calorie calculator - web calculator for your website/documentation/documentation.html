Introduction

Welcome to our Calorie Calculator, This Calculator to help you achieve your health and fitness goals with precision and ease. Whether you're looking to lose fat, maintain your current weight, or gain muscle mass, our calculator is designed to give you the right nutrition you need.

The calculator is built using HTML, CSS, and JavaScript libray Jquery.

Quick Start

Copy-paste the stylesheet link into your head before all other stylesheets to load our CSS.


<link href="css/Calc-style.css" rel="stylesheet" type="text/css">

Copy-paste the JavaScript link into your body bottom all other after Html element.


<script src="js/jquery.js"></script>
<script src="js/Calc-script.js"></script>

Copy-paste the .ms-cl class into your body Element.

<html>
    <body>
        <div class="ms-cl">
            //write Your Html code
        </div>
    </body>
</html>

Features

Features
Instantly calculate your daily calorie needs based on your unique profile, including gender, age, height, weight and activity level.
Select your fitness goal and let the calculator adjust calorie intake accordingly.
Get a detailed breakdown of your protein, fat and carbohydrate needs in both calories and grams.

How To Use (HTML Structure)

HTML Structure Description
.ms-cl The main container for the calculator.
.container , .cl-wrapper, .main-calc The container & cl-wrapper containing the input fields and output sections.
.cl-left-div This Section displaying User input field section.
.cl-right-div This Section displaying output section.
.unit_check Allows the user to choose between metric and US units for height and weight.
.slider-output-section Displays calculated daily calorie and gram requirements for protein, fat and carbohydrates.

Form Inputs

This HTML document appears to be a Calorie Calculator, allowing users to input their Calorie.

Inputs Description
Gender - () Choose your gender (Male or Female).
Age - (#cl_age) Enter your age.
Unit - (.unit_check) Select the measurement units you want to use for your length and weight.
Length (cm) - (#cl_height_1) Input your Length in centimeters.
Weight (kg) - (#cl_weight_1) Input your weight in kilograms.
Length (feet) - (#cl_height_2) Input your Length in feet.
Length (inches) - (#cl_height_22) Input your Length in inches.
Weight (lbs) - (#cl_weight_2) Input your weight in pounds.
Activity level - (#cl_activity) Select your Activity level of physical activity.
Goal - (#cl_goal) Choose your goal.

Additional Notes

  • Each activity level is clearly defined to help users accurately assess their general physical activity, ensuring more accurate caloric need estimates.

  • The calculator is designed with a clean and intuitive interface to make it easy for users to input their data and understand the results.
