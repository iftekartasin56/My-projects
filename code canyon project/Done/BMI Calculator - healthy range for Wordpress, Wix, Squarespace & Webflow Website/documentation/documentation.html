Introduction

Welcome to the BMI Calculator, a powerful and versatile tool designed to help users quickly and accurately calculate their Body Mass Index. This documentation provides detailed information on how to integrate and utilize the BMI Calculator within your Website, ensuring a seamless user experience.

The calculator is built using HTML, CSS, and JavaScript.

Quick Start

Copy-paste the stylesheet link into your head before all other stylesheets to load our CSS.


<link href="css/Calc-style.css" rel="stylesheet" type="text/css">

Copy-paste the JavaScript link into your body bottom all other after Html element.


<script src="js/Calc-script.js"></script>

Copy-paste the #bmi__calculator id into your body Element.

<html>
    <body>
        <div id="bmi__calculator">
            //write Your Html code
        </div>
    </body>
</html>

Customization

If you want to customize the BMI calculator, there are several aspects you can modify, including the appearance, functionality, and calculation logic. Here are a few customization ideas:

Customization Description
Styling (CSS) Modify the styles in the "Calc-style.css" file to change the appearance of the BMI calculator. You can adjust colors, fonts, spacing, and layout to match your desired design.
Unit Conversion Adjust the default values in the input fields for height and weight in both US and Metric units to suit your preferences.
BMI Ranges and Descriptions Customize the BMI ranges and descriptions in the HTML section where the BMI graphic is defined. You can change the cutoff values and the corresponding descriptions for each BMI category.
Result Messages Modify the messages in the BMI details result text sections to better suit your audience or provide more specific advice
Weight Range Calculation Adjust the calculation for the healthy weight range based on height. Currently, the BMI calculator provides a static range of "0 - 0" which doesn't seem accurate. You might want to implement a more accurate calculation or fetch this information dynamically.
JavaScript (Calc-script.js) Customize the JavaScript functions for unit conversion, BMI calculation, and result display. You can modify the logic to better fit your requirements or enhance the user experience.
Localization If your audience speaks different languages, consider adding localization support by translating text content and messages.
Responsive Design Ensure that the BMI calculator is responsive to different screen sizes. You may need to adjust styles or layout in the CSS file for better compatibility with various devices.

Form Inputs

This HTML document appears to be a BMI (Body Mass Index) calculator, allowing users to input their height and weight to calculate their BMI.

Inputs Description
Unit Selection - Two options for units : "US Units" and "Metric Units."
- Clicking on these options triggers a function (bmi__calculator.changeUnit()) to switch between the two unit systems.
Height Input - For "US Units" : Two input fields for feet (#bmi__height_ft) and inches (#bmi__height_in).
- For "Metric Units" : One input field for centimeters (#bmi__height_cm).
Weight Input For both unit systems : Two input fields for weight : - Pounds (#bmi__weight_lbs), - Kilograms (#bmi__weight_kg).

Additional Notes

  • Unit Switching - The unit switching functionality is implemented efficiently using the changeUnit method.

  • Input Validation - The script includes some basic input validation in the event listeners, such as limiting the input values for height and weight.

  • BMI Marker Position - The script adjusts the position of the BMI marker on a graphical scale based on the calculated BMI.

  • Readable Code - The code is relatively readable and well-organized. Variable names are descriptive, and the code structure makes it easy to understand the logic behind unit switching, input validation, and BMI calculations.
