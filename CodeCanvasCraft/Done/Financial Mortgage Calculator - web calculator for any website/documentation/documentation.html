Introduction

Welcome to our Financial Mortgage Calculator. Our calculator is designed to help you estimate your monthly mortgage payments based on various loan amounts, terms, and interest rates. With our calculator, you can easily compare different mortgage options and choose the one that best suits your needs.

The calculator is built using HTML, CSS, and JavaScript, and it also utilizes the popular JavaScript library jQuery and charts.

Quick Start

Copy-paste the stylesheet link into your head before all other stylesheets to load our CSS.


<link href="css/Calc-style.css" rel="stylesheet" type="text/css">

Copy-paste the JavaScript & jQuery link into your body bottom all other after Html element.


<script src="js/Calc-script.js"></script>
<script src="js/jquery.js"></script>
<script src="js/chart.js"></script>

Copy-paste the .ms-cl class into your body Element.

<html>
    <body>
        <div id="ms_cl">
            //write Your Html code
        </div>
    </body>
</html>

How To Use (HTML Structure)

The Financial Mortgage Calculator's HTML structure consists of various sections and elements to handle different functionalities. The main sections include:

HTML Structure Description
.ms_cl The main container for the calculator.
.cl-wrapper The wrapper containing the mortgage input fields and output sections.
.switch-sec The section displaying the "Annual Property Taxes, Annual Homeowners Insurance, Monthly Maintenance Fee" Switch Icon.
.btn-sec The Button Calculate after put the input data.
.rs_sec The section displaying the estimated monthly payment.

Customization

The Financial Mortgage Calculator's appearance and styles are fully customizable using CSS. The CSS classes can be modified to change colors, fonts, and other visual elements to match the design of your website.

Form Inputs

The following form inputs are available for the user to enter their mortgage details:

Inputs Description
Home Price (#in_1) The total price of the home.
Down Payment (#in_2) The down payment amount(Dollar or percentage).
Loan Term (#in_4) The loan term in years (e.g-5, 10, 15, 20, 25, 30 years).
Interest Rate (#in_5) The annual interest rate for the mortgage.
Annual Property Taxes (#in_6) The annual property taxes.
Annual Homeowners Insurance(#in_7) The annual homeowners insurance.
Monthly Maintenance Fee(#in_8) A monthly maintenance fee is a fee charged by a financial institution.

Additional Notes

  • The Financial Mortgage Calculator is designed for educational and estimation purposes only. It provides approximate values and should not be considered as a financial offer or approval.

  • Mortgage calculators usually assume that the interest rate is fixed and constant throughout the loan term.

  • Mortgage calculators can help you plan your budget and savings goals, but they are not a substitute for professional advice.
