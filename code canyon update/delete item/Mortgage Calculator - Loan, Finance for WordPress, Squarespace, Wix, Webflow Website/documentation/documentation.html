Introduction

The "Mortgage Calculator" is a powerful and user-friendly web tool designed to help users estimate their mortgage payments for a home loan. It provides a simple and efficient way to calculate monthly payments based on various factors, such as home price, down payment, interest rate, loan term, annual property taxes, and annual home insurance.

The calculator is built using HTML, CSS, and JavaScript, and it also utilizes the popular JavaScript library jQuery and the AutoNumeric library for number formatting.

Quick Start

Copy-paste the stylesheet link into your head before all other stylesheets to load our CSS.


<link href="css/Calc-style.css" rel="stylesheet" type="text/css">

Copy-paste the JavaScript & jQuery link into your body bottom all other after Html element.


<script src="js/Calc-script.js"></script>
<script src="js/jquery.js"></script>

Copy-paste the .ms-cl class into your body Element.

<html>
    <body>
        <div class="ms-cl">
            //write Your Html code
        </div>
    </body>
</html>

Features

Features Description
Estimate Monthly Mortgage Payment Users can input essential information like home price, down payment, interest rate, and loan term to calculate their estimated monthly mortgage payment.
Interest & Principal Breakdown Users can view the detailed breakdown of interest and principal payments over the loan term.
Yearly Payment Schedule Users can select a specific year to view the remaining balance, principal paid, and interest paid for that particular year.
Responsive Design The calculator is designed to work smoothly across different devices and screen sizes.

How To Use (HTML Structure)

The Mortgage Calculator's HTML structure consists of various sections and elements to handle different functionalities. The main sections include:

HTML Structure Description
.ms-cl The main container for the calculator.
.cl-wrapper The wrapper containing the mortgage input fields and output sections.
.middle-sec The section displaying the "Interest & Principal Breakdown" toggle button.
.cl-input The input fields for entering the mortgage-related data.
.output-sec The section displaying the estimated monthly payment and additional cost breakdown.
.sp-table-border The section displaying the summary of costs in a table format.

Customization

The Mortgage Calculator's appearance and styles are fully customizable using CSS. The CSS classes can be modified to change colors, fonts, and other visual elements to match the design of your website.

Form Inputs

The following form inputs are available for the user to enter their mortgage details:

Inputs Description
Home Price (#home_price) The total price of the home.
Down Payment (#down_payment) The down payment amount.
Down Payment Percentage (#down_payment_percentage) The down payment percentage.
Interest Rate (#interest_rate) The annual interest rate for the mortgage.
Loan Term (#loan_term) The loan term in years (e.g., 30, 25, 20, 15, 10 years).
Annual Property Taxes (#annual_property_taxes) The annual property taxes.
Annual Home Insurance (#annual_home_insurance) The annual home insurance.
Choose Year (#choose_year) Choosing Year.

Calculations

The Mortgage Calculator performs the following calculations:

Method Description
Monthly Payment The calculator estimates the monthly mortgage payment based on the provided input data, including home price, down payment, interest rate, and loan term. It also considers property taxes, home insurance, and potential Private Mortgage Insurance (PMI) costs if applicable.
Interest & Principal Breakdown The calculator calculates and displays the breakdown of interest and principal payments over the loan term.
Yearly Payment Schedule Users can select a specific year from the drop-down list to view the remaining balance, principal paid, and interest paid for that particular year.

Additional Notes

  • The Mortgage Calculator is designed for educational and estimation purposes only. It provides approximate values and should not be considered as a financial offer or approval.

  • Calculator results are rounded down to the nearest dollar.

  • Users should contact financial institutions or mortgage professionals for precise estimates and further assistance.
