Introduction

Welcome to our Pre and Post Money Valuation Calculator, Pre and Post Money Valuation Calculator helps entrepreneurs, investors and financial analysts assess the value of a company before and after receiving an investment.

This calculator helps understand how investments affect a business's valuation, helping to make informed decisions about funding and equity allocation.

The calculator is built using HTML, CSS, and JavaScript libray Jquery and Autonumeric Number format.

Quick Start

Copy-paste the stylesheet link into your head before all other stylesheets to load our CSS.


<link href="css/Calc-style.css" rel="stylesheet" type="text/css">

Copy-paste the JavaScript link into your body bottom all other after Html element.


<script src="js/jquery.js"></script>
<script src="js/autonumeric.js"></script>
<script src="js/Calc-script.js"></script>

Copy-paste the .ms-cl class into your body Element.

<html>
    <body>
        <div class="ms-cl">
            //write Your Html code
        </div>
    </body>
</html>

Features

Features
Calculates the company's pre-money valuation based on the investment and equity percentage paid.
Calculates the post-money valuation of the company by adding the investment to the pre-money valuation.

HTML Structure Description

.em-context This em-context containing the input fields and output sections.
.mid_container This mid_container containing the input fields.

Form Inputs

This HTML document appears to be a Pre and Post Money Valuation Calculator, allowing users to input their Pre and Post Money Valuation.

Inputs Description
Investment - (#in_1) Enter the amount of money invested in the company.
Investor's equity - (#in_2) Enter the percentage of equity the investor will receive in return for their investment.

Additional Notes

  • Ensure that the investment amount and investor's equity percentage are entered correctly to get the correct valuation.

  • Pre-money and post-money values ​​are updated in real-time as you input data, allowing for immediate feedback and adjustments.

  • This calculator is useful for startup founders, investors and financial analysts to quickly assess the financial impact of potential investments.

  • 100% Responsive design
